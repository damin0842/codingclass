반복문

반복문은 프로그램에서 필요한 결과 값을 도출하기 위해 실행문의 순서를 반복적으로 실행시키는 문법을 말합니다.
반복문에는 while문, do while문,for문이 있으며, for문을 가장 많이 사용합니다.

for ( 초기값; 조건식; 증감값){
    실행문
}

01. 0부터 99까지 출력하기

for( let i=0; i<100; i++){
    document.write(i)
}

for( let i=2; i<=100; i+=2){  // i + 2 --> i = i + 2

07. for문을 이용해서 테이블 출력하기

중첩 for문을 이용해서 테이블을 만들 수 있습니다.

//for문을 이용해서 테이블 만들기
let table = "<table class='testTable'>";

//10번 반복(for문)
for(let i=1; i<11; i++){
    table += "<tr>";
    for(let j=1; j<11; j++){
        let sum = i*j;
        table += "<td>"+sum+"</td>"; 
    }
    table += "</tr>";
}
table += "</table>";
document.write(table);
